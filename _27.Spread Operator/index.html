27.Spread Operator

Spread syntax allows you to expand iterable elements (such as arrays or strings) in specific contexts:


For functions: where the expected number of arguments for function calls is zero or more.
For array literals: to include elements from another array or iterable.
For object expressions: in places where the number of key-value pairs is zero or more (for object literals).





The Object.assign() method is used to copy the values of all enumerable own properties from
one or more source objects to a target object. After copying, it returns the target object.
