44.Descriptors

Object.keys()

The Object.keys() method returns an array of a given object's own enumerable properties,
in the same order as they would be iterated over by a for...in loop. The difference is that
a for...in loop enumerates properties from the prototype chain as well.

Syntax:
Object.keys(obj)



Object.values()

The Object.values() method returns an array of the values of a given object's enumerable
properties, in the same order as they would be iterated over by a for...in loop. The difference
is that a for...in loop also enumerates properties from the prototype chain.

Syntax:
Object.values(obj)



Object.entries()

The order of elements in the array returned by Object.entries() does not depend on how the object is defined.
If a specific order is needed, the array must be sorted before using the method, for example:
Syntax:

Object.entries(obj)
