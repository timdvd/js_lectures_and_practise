41.Optional chaining operator ?.

optional chaining operator ?.



The optional chaining operator ?. allows you to access the value of a property at any level of nesting in a chain of interconnected objects without
having to check each intermediate property for existence. The ?. operator works similarly to the . operator, except that it does not throw an
exception if the object whose property or method is being accessed is null or undefined. In such cases, it returns undefined.
